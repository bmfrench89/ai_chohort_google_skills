# Assignment 9 — Observability in Google Cloud

Module 9 of the AI cohort. Two Google Cloud Skills Boost courses covering the full
observability stack: metrics, dashboards, uptime checks, alerting, SLIs/SLOs with error
budgets, and distributed tracing with Cloud Trace.

**Full rubric + progress tracker:** [`rubric.md`](rubric.md)

## How this one differs from Assignment 8

These are **guided labs** (step-by-step, mostly Cloud Console UI), not unguided challenge
labs. So this folder tracks **evidence and progress** rather than solution artifacts:
- Each lab's *Check my progress* checkpoints auto-grade as you go.
- Evidence = **screenshots** of each finished lab (labs may not mint a badge) → saved under
  each course's `screenshots/`.
- Final submission = badge URLs (if any) + your **public profile URL**, collected in
  [`submission.md`](submission.md).
